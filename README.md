# Elite Expense Tracker

A modern **desktop personal finance manager** built with **React + Electron**.

Elite Expense Tracker helps users manage their **expenses, income, investments, and credit card dues** from a single clean dashboard while keeping all data **private and stored locally**.

---

## Key Features

• Track and categorize daily expenses  
• Monitor income and available balance  
• Manage cash and bank accounts  
• Track investment values  
• Credit card due reminders  
• Financial health insights through charts  
• Fully offline desktop application  
• Secure local data storage (no cloud dependency)

---

## Tech Stack

Frontend  
- React  
- Vite  

Desktop Framework  
- Electron  

UI & Visualization  
- TailwindCSS  
- Chart.js  

---

## Application Architecture

Elite Expense Tracker follows a **desktop web hybrid architecture**:

React UI → Handles user interface and state management  
Electron → Wraps the application as a cross-platform desktop app  
Local Storage → Stores all financial data securely on the user's machine  

This ensures **fast performance, offline capability, and user privacy**.
